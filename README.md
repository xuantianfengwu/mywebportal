# MyWebApp

#### 介绍
XueZhihuan's MyWebApp

#### 软件架构
软件架构说明
- 根目录和backend文件夹是Flask的后端
- frontend文件夹是React的前端

#### 安装教程
1.  在根目录运行python my_app.py可以启动后端
2.  在frontend文件夹运行npm run start启动前端网页
3.  在前端网页上进行产品功能的使用
